💡 Copy and paste the command below to install everything and run the tool:
💡 คัดลอกและวางคำสั่งด้านล่างนี้เพื่อติดตั้งสิ่งที่จำเป็นทั้งหมดและเริ่มรันโปรแกรมทันที:
